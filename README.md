# Upload a file to Amazon S3

This project is to uploading files to Amazon S3 for `[gongheung.com](https://gongheung.com)`.

Includes:
- server
  - Generate S3 pre-signed url on Amazon Lambda
- client
  - Get a pre-signed url and upload a file
